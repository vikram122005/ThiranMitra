🚀 Recent Feature Updates (V2 Enhancements)

1. Smart Job API Filtering & Datasets
Extensive dummy dataset built directly into the local fallback system containing **30 high-quality jobs** spanning 6 major sectors (Tech, Banking, Marketing, Logistics, Design, HR).
- Local search filtering across:
   - Location
   - Exact Category Match
   - Salary Scales
   - Work Mode & Job Type

2. Genuine Authentic Government Schemes
- The platform now serves **30 real-world government schemes** spanning 6 focus areas: 
   - Skill Development (PMKVY, DDU-GKY, Skill India Hub)
   - Employment (NCS, Rozgar Mela, PMEGP)
   - Business Loans (MUDRA, Stand-Up India, PM SVANidhi)
   - Women (Mahila Samman, Beti Bachao)
   - Rural (MGNREGA, PM-KISAN)
   - Students (NSP, SWAYAM, Pragati)
- Implemented **true external application routing**—clicking "Learn More & Apply" directs candidates straight to the official `.gov.in` / `.nic.in` portal.

3. Dynamic Algorithm-Driven Course Recommendations
Skill gap analysis now outputs direct, functional, externally routed links to upskilling platforms.
- If a user lacks "Python", the system will instantly construct a learning path mapping them to relevant courses on:
   - YouTube
   - Udemy
   - Coursera
   - SWAYAM
   - NPTEL
- Action buttons natively open in new window allowing candidates to seamlessly bridge the gap effortlessly.
